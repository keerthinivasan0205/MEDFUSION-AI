# MEDFUSION-AI

An AI-powered multi-disease prediction system built with Flask, MySQL, TensorFlow, and ReportLab.

## Tech Stack
- Flask (REST API)
- MySQL + SQLAlchemy ORM
- Flask-JWT-Extended (Authentication)
- TensorFlow (.h5 models)
- ReportLab (PDF Report Generation)
- Flask-CORS

## Supported Disease Predictions
- Skin Disease Detection
- X-Ray Analysis
- MRI Analysis
